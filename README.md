# 🧑‍💻 Uktam Turgunov — Professional Portfolio

Personal portfolio website built with modern web technologies, focusing on clean UI/UX and dynamic data rendering.

🔗 **Live Demo:** [oktamturgun.github.io/my_portfolio](https://oktamturgun.github.io/my_portfolio/)

---

## ✨ Key Features

- **🌓 Dynamic Theme Switching:** Fully functional Dark and Light modes with persistent user preference (Local Storage).
- **📊 Dynamic Content:** Projects are dynamically rendered from a `projects.json` file using asynchronous JavaScript (`fetch` API).
- **📱 Responsive Design:** Optimized for all devices using Bootstrap 5 and custom media queries.
- **🚀 Performance:** Optimized assets and smooth scroll-reveal animations for an engaging UX.
- **✉️ Contact Integration:** Integrated contact form with EmailJS for direct communication.

---

## 🛠 Tech Stack

- **Frontend:** HTML5, CSS3 (Custom Variables), JavaScript (ES6+)
- **Frameworks:** Bootstrap 5
- **Icons:** Font Awesome 6
- **Data:** JSON (External data fetching)

## 🚀 Development & Local Run

1. Clone the repository:
   ```bash
   git clone https://github.com/OktamTurgun/my_portfolio.git
   ```

2. Open with a local server (e.g., Live Server in VS Code) to allow `fetch()` API to load `projects.json` correctly.
